Political Representation in R

The R package polrep implements many different measures of political representation, covering both descriptive representation (unordered groups) and substantive representation (issue positions).

The package is currently in early development(alpha), but the functions included should be fairly useable. The main reason to designate this package early development is that new functions will be added (i.e. it is not feature complete at this stage). By now function names should be fairly set.

The R package polrep is open source distributed under the GNU General Public License (GPL). It is written and maintained by Didier Ruedin.

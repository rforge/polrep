Political Representation in R

The R package polrep implements many different measures of political representation, covering both descriptive representation (unordered groups) and substantive representation (issue positions).

The package is currently in early development, but the functions included should be fairly useable.

Installation and Use

Installation

To install the R package polrep, use the following command within R:

install.packages("polrep", repos="http://R-Forge.R-project.org")

General Note on Using the Functions

Almost all measures in this package require two vectors as input, one for the population (citizens) and one for the representatives (legislators). These vectors should be in the form of frequency vectors, that is giving the number of observations at each position. In the following I refer to these as Z (for the citizens or population) and R (for the representatives or legislators).

The helper functions collapse and expand can be used to create frequency vectors.

At this stage, please refer to the help files in R on further details on the different measures of political representation.

Individual Representation Scores

Calculates individual representation scores. Use: irs(Z,R) for continuous data and irs.cat(Z,R) for categorical data. Using collapse() and then irs.cat() is significantly faster than running irs() if you have categorical data.

For a full description of the concept of individual representation scores, refer to Ruedin, D. Individual representation: A different approach to political representation. Representation 48, no. 1 (2012): 115-29.

Gallagher index

The Gallagher index of least squares is available as Gallagher(Z,R). Use Gallagher.1(Z,R) to have the values inverted so that higher values stand for greater proportionality.

Farina index

Calculates the Farina index or cosine proportionality measure. Use: Farina(Z,R).

Rose index

The Rose index is 1-LHI. Higher values stand for greater proportionality. Use: Rose(Z,R)

Rae index

Use: Rae(Z,R); or Rae.1(Z,R) to have the values inverted to give higher values for greater proportionality.

Sainte-Laguë

Use: st.lague(Z,R)

The inverted Sainte-Laguë is also available: st.lague.inverted(Z,R)

Effective Number of Groups

Uses the Laakso and Taagepera approach to the effective number of parties. Use: neff(x).

The alternative (rarely used) measure, based on the product is available as neff.1(x).

Heterogeneity

Use: H(x)

Others

There are other measures included, many of which more conceptual rather than in actual use. At this stage, refer to the built-in documentation for further information.

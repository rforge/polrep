Political Representation in R

The R package polrep implements many different measures of political representation, covering both descriptive representation (unordered groups) and substantive representation (issue positions).

The package is under development and new functions will be added (i.e. the package is not feature complete at this stage). All the functions included should be fairly useable and stable.

The R package polrep is open source distributed under the MIT License. It is written and maintained by Didier Ruedin.
